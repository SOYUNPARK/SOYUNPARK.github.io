### 커밋에 이름표 붙이기: tag

| 명령어                   | 기능                    |
| ------------------------ | ----------------------- |
| git tag                  | 태그조회                |
| git show [태그명]        | 태그 정보 확인          |
| git tag [태그명]         | 태그 추가               |
| git push origin [태그명] | 원격 repo에 태그 올리기 |
| git tag -d [태그명]      | 태그 삭제               |
